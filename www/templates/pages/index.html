Volunteer computing of WWW stats

  1. You can start by reading more on the project here or if you know everything already, continue with the step 2.

  2. All you have to do is download this extension

    Install SpeedTracer

  3. And then open a few tabs by clicking on the button below

    Start Measuring

  • What is Speed Tracer?

    Speed Tracer is an extension for Chrome web browser provided by Google that helps measure the inner operations performed by the browser. Find out more about it here.

  • Can I see the results?

    You will able to see the results as soon as there are enough of them. However you will not be able to see your individual results as they are not marked in any vay to be identified as yours.

  • What is the goal of this?

    Discovery of the common coding patterns that slow down web page rendering and to shut down or efficiently reuse at least one power plant by avoiding these patterns.
